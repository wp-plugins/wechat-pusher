== Description ==

微信推送助手是我制作的用于帮助微信公众号推送文章的Wordpress插件，现有的插件鲜有此功能，便自制之。欢迎拍砖，问题咨询QQ493554781，或者手机13691579267。

1. 上传到 `/wp-content/plugins/` 目录
2. 然后设置
3. 详细设置及问题解答请至http://pq.orgwis.com/?p=50页面
